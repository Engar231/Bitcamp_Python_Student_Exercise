This repository contains a Python exercise developed during the Bitcamp course, focused on managing student-related data and logic.
The project was designed to practice Python fundamentals, including variables, control flow, functions, basic data structures, and introductory object-oriented programming concepts.

The application implements simple logic to handle student information and interactions through Python scripts, with an emphasis on clean code, readability, and structured problem solving.

This repository represents an educational exercise aimed at strengthening core Python skills and building a foundation for more advanced applications.
